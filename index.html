About PIP Mode

You can enable PIP mode by clicking on its icon on the top right. In supported desktop browsers, it will activate Picture-In-Picture mode.
If you are on a mobile device, clicking on that button will activate fullscreen mode. In order to switch to PIP mode, just go to your device's homescreen without exiting fullscreen. If your browser is supported then PIP mode will be activated.
If your desktop browser does not support Picture-In-Picture API, like Firefox, then it will open fullscreen as well.

Welcome to Tomodoro!

Tomodoro is an open source pomodoro web app with offline support and PIP mode.


About PIP mode

Tomodoro supports Always on Top mode, or Picture-In-Picture mode, right in the browser, without having to install any electron app.
PIP mode is supported on Chromium based browsers(Tested on desktop and android Chrome).
Safari not tested (yet) but might work.
Desktop Firefox is not supported.


PIP mode on mobile devices

If you are on a mobile device, clicking on that button will activate fullscreen mode. In order to switch to PIP mode, just go to your device's homescreen without exiting fullscreen. If your browser is supported then PIP mode will be activated.
